title: MetingJs 服务器
date: 2022-10-08 17:03:27
---

Danmu.Server 已合并 MetingJS.Server，可作为 MetingJs 的服务端使用。

## API

- `https://danmu.u2sb.com/api/meting?server=:server&type=:type&id=:id&r=:r`

## 示例

- `https://danmu.u2sb.com/api/meting?server=tencent&type=song&id=003UkWuI0E8U0l&r=0.8074837167524542`
